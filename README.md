# Calculator Web App

A responsive web-based calculator built to perform basic arithmetic operations with a clean and intuitive user interface.

## Overview

This project is a lightweight calculator application designed to perform standard arithmetic operations including addition, subtraction, multiplication, and division.

The goal of this project was to strengthen my understanding of frontend development fundamentals, including DOM manipulation, event handling, and responsive UI design.

---

## Features

- Perform basic arithmetic operations (+, −, ×, ÷)
- Clear and reset functionality
- Responsive design for mobile and desktop
- Input validation for improved user experience
- Real-time display updates

---

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Key Learning Outcomes
